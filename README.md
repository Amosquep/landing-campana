# 🌙 Luna Íntima - Landing Page

Landing page moderna para una campaña informativa/comercial de lencería, desarrollada con HTML, CSS y JavaScript. El proyecto aplica buenas prácticas de organización, diseño responsive y experiencia de usuario.

---

## 🚀 Demo

Puedes ver el proyecto en línea aquí:

👉 https://Amosquep.github.io/landing-campana/

---

## 🧠 Descripción del proyecto

Esta landing page fue diseñada para simular una campaña de una marca de lencería moderna, enfocada en:

* Elegancia visual
* Experiencia de usuario (UX)
* Diseño responsive (adaptable a móvil)
* Interacción básica con JavaScript

Incluye secciones típicas de marketing digital como:

* Hero (mensaje principal)
* Colección de productos
* Beneficios
* Oferta promocional
* Formulario de suscripción

---

## 🛠️ Tecnologías utilizadas

* HTML5 → estructura del sitio
* CSS3 → estilos y diseño moderno
* JavaScript → interacción y eventos
* GitHub → control de versiones
* GitHub Pages → despliegue web

## 🎯 Buenas prácticas aplicadas

* Separación de responsabilidades (HTML, CSS, JS)
* Uso de clases reutilizables
* Organización por carpetas
* Código limpio y legible
